# Notebook2REST_project

This github page is used for a exercise of the DevOps and Cloud-based software course from the UvA. The goal is to translate notebooks to REST and put it inside the cloud. In this case, the cloudservice used is AWS.

The project exists of a part that extracts the parameters, a fastAPI part, a docker part and a part that integrates everything in AWS.

The team members are: Jona Aalten, Mike van der Deure, Theo Gatea and Colin de Koning.

## Roles:

Jona is the product manager and has the task to keep the overview over the project. He had the task to run AWS.
Mike and Theo has the task to translate the notebook to REST and to create the fastAPI part that would later be connected to the cloud and docker part.
Colin had the task to handle the part about docker.
